# Shapes Graphs

This folder contains the macro-level shapes graphs for validating Transformation Manifest Files (TMFs). Micro-level shapes graphs are co-located with their respective refactoring actions in the [Refactoring Actions](../RefactoringActions/README.md) folder.

## Macro-level shapes graphs

| Shapes Graph | Transformation Process |
|---|---|
| [ShapeEvolution](ShapeEvolution/README.md) | Ontology Evolution |
| [ShapeLocalisation](ShapeLocalisation/README.md) | Ontology Localisation |
| [ShapeRefactoring](ShapeRefactoring/README.md) | Ontology Refactoring |
| [ShapeTransformation](ShapeTransformation/README.md) | Ontology Transformation |
