# Composite Action

**Definition type:** OWL Class \
**Fragment identifier:** `fract:CompositeAction` \
**URI:** `https://w3id.org/refract#CompositeAction`

A [local action](LocalAction.md) composed of two or more [atomic actions](AtomicAction.md), representing a batch of changes in a single action.

For the list of composite actions, see [Local Action](LocalAction.md).

`owl:Thing` → `rdfs:Class` → `prov:Activity` → `fract:RefactoringAction` → `fract:LocalAction` → `fract:CompositeAction`
